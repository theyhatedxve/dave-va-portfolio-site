# Christian Dave Tagadiad VA Portfolio Website

A Vite React + JavaScript portfolio website based on the uploaded portfolio PDF.

## Tech stack

- Vite
- React
- JavaScript
- CSS

## Features

- Responsive navigation
- Hero, About, Services, Tools, Process, Case Studies, Results, Testimonials, Packages, Why Me, and Contact sections
- Interactive case study tabs
- Contact form with mailto inquiry flow
- Extracted image assets from the PDF
- Mobile responsive layout

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Use these settings:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
